# 👁️ Real-Time Face Detection using OpenCV

## 📌 Project Overview
This project performs real-time face detection using a webcam. 
It uses OpenCV and Haar Cascade classifiers to detect human faces in live video feed and draws bounding boxes around them.

---

## 🧠 Technologies Used
- Python
- OpenCV (cv2)
- Haar Cascade Classifier

---

## ⚙️ How It Works
1. Captures live video from webcam
2. Converts each frame to grayscale
3. Detects faces using Haar Cascade model
4. Draws rectangles around detected faces
5. Displays real-time output
